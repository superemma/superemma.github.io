Lupus co-occurence network

Welcome to this interactive network,you can start your exploration by adjusting the settings in the bottom. Here are some basics:

1. To make the labels bigger, increse the font size and the stroke width.

2. You may also adjust the spring value and other network physics.

3. To stop the network from tremoring, uncheck the "enabled" in the physics settings. After that you can drag the nodes anywhere in the static figure.

4. You can also select only the nodes/edges of interest by using the drop down menu below.

5. If you feel you messed up, don't panic! Just refresh the webpage and everything will reload.

Have fun!
